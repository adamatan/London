About

Serverless Days TLV is developer-oriented conference about Serverless tech. We believe in show, don't tell sessions, in which developers share their experience and lessons from actual projects.

We will host sessions about FaaS, architecture, security, performance and monitoring Serverless applications with the best speakers we can find.

Serverless Days Tel Aviv is about content for seasoned cloud developers and entrepreneurs. We will skip introductory tutorials and hello, worlds and focus on lessons learned by local and international users with extensive experience.

Serverless Days TLV is member of Serverless Days (AKA JeffConf), a family of events around the world fostering communities around serverless tech. We follow and enforce the Serverless Days Code of Conduct.

Registeration is now open. The ticket price is 50₪, which are donated to The Public Knowledge Workshop.

Call for Papers

We welcome local and international speakers, and sessions about any Serverless topic, including:

  • Stories from the trenches: your triumphs and failures with serverless projects
  • Serverless tools and techniques
  • Performance
  • Monitoring
  • Security
  • Open source projects, either using serverless or featuring serverless tools
  • The serverless marketscape
  • The human factor: the development model in serverless architecture
  • Storage and DBs for Serverless architecture

The Call For Papers is open till May 3rd, 23:59:59 UTC.
